FactSphere Chrome extension

These settings are used on Google Search pages to call your local FactSphere backend and show average hallucination risk.

Google Programmable Search (optional)

Use the same credentials as in FactSphere’s web settings: a Custom Search JSON API key and your Search Engine ID (cx) from Programmable Search Engine. Both fields are required for Google API results; leave blank to use only Bing / DuckDuckGo / Wikipedia.

Load the extension from FactSphere/chrome-extension, start python search_backend.py, then run a Google search. The floating badge uses the same /search API as the FactSphere web UI.
